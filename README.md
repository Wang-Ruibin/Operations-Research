## 📖 项目简介

本项目为运筹学课程设计的算法实现集合，涵盖了**图论**与**网络流**领域的经典算法。所有代码均使用 C++ 编写，基于邻接表存储结构，适用于洛谷（Luogu）在线评测平台的对应题目。

## 🔑 关键算法解析

### Dinic 最大流

```
时间复杂度：O(n² · m)
核心优化：BFS 构建分层图 + DFS 多路增广 + 当前弧优化
```

### 最小费用最大流（MCMF）

```
时间复杂度：O(n · m · maxflow)
核心思想：每次沿最短增广路增广，保证费用最优
```

### Johnson 全源最短路

```
时间复杂度：O(n · m · log n)
核心思想：引入虚拟源点，通过势函数消除负权边，再对每个起点跑 Dijkstra
```
